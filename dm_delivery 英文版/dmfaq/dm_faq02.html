納期 ・発送 ・返品について
Q1 発送から到着まで
  A1 変申し訳有りませんが、到着日指定はお受け出来ません。通常（発送日-到着）までは6-7日程度ですが、地理的/気象条件・交通事情・繁盛期等により前後する場合があります。お早目のご準備をお願い致します。
     
Q2 到着しなかったDMはどうなるの？
  A2 未着分のDMは、一旦弊社へ戻ります。未着分に関しては通常2週間程度ですべて戻ります。
その後は返却物を御社へ発送するか、弊社にて無償で処分するかをお選び下さい。
また、その場合未着分の送料は返金出来ませんのでご了承下さい。
    宛名印刷の返送先住所の挿入
    a. 弊社にて無償で処分 『 The undeliverable exempts returns 』 推奨
    b.返却物を御社へ発送 『Undelivered Return to : P.O.Box 5-183, Taichung,Taiwan 』（当社私書箱）
     
Q3 返品、返品期限
  A3 当サービスは商品を返却して代金を返すことができません
     
Q6 海外からの発送でちゃんと届くか不安
  A6 Mailbinから台湾、香港の郵便局へ局だしを行う際は、常にバルクメール袋（郵便物をまとめて送付するための布製の大きな袋）を使用し局だしを行います。日本の国際郵便局（東京、大阪等）へはこの袋に全てのDMがまとめてはいった状態で送付されます。従ってこのバルクメール袋ごと紛失するといったことが起こらない限り、台湾、香港と日本間での不着は発生しません。安心してお申し込みください。
 
Q7 日本以外へのDM発送は可能？
  A7 可能です
  重量 = DM + 封筒 + 宛名

・北米 : 65.5円 ・中南米 : 65.5円
・ヨーロッパ : 65.5円 ・アフリカ : 65.5円
・アジア : 55円  
・ オーストラリア / ニュージーランド : 55円
    上記を超える場合は
   
郵便料金 重量 > 20g  
・北米 ( + 35円 / 20 g ) ・中南米 ( + 35円 / 20 g )
・ヨーロッパ ( + 35円 / 20 g ) ・アフリカ ( + 35円 / 20 g )
・アジア ( + 24.5円 / 20 g )  
・ オーストラリア / ニュージーランド ( + 24.5円 / 20 g )
   
到着の営業日

Q8 ３枚以上のDM原稿は送付できませんか？
  A8 DM原稿が３枚以上になった場合、重量の関係で郵便料金の区分が高くなってしまいます。従って弊社の標準コースは原稿２枚以内でご案内しています。３枚以上でも別途対応させていただくことは可能です。お問い合わせください

Q11 7営業日以内にDM発送とはいつから計算？
